# Web Enumeration with Nikto

## Overview

Nikto was used within the SOC homelab environment to perform web enumeration and vulnerability reconnaissance against the DVWA web server.

The activity simulated common attacker reconnaissance behavior used to identify:

* exposed services
* insecure configurations
* outdated software
* missing security controls
* potentially vulnerable web resources

The scan generated telemetry that could be monitored and investigated within:

* Apache access logs
* Suricata IDS
* Wazuh SIEM

---

# Objective

The primary goals of this exercise were:

* perform web server enumeration
* identify exposed web services
* generate reconnaissance telemetry
* validate SIEM visibility
* test IDS monitoring capabilities
* support detection engineering workflows
* simulate realistic attacker behavior
